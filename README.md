Project: genAICodex
=================

## Configuración

Antes de ejecutar la aplicación, asegúrate de configurar las siguientes variables de entorno:

file: `.env`

```bash
    OPENAI_API_KEY=
    LANGCHAIN_API_KEY=
```

## Ejecución

Para ejecutar la aplicación, simplemente usa el siguiente comando:

```bash
streamlit run main.py
```
